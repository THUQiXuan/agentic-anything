# code CLI

Zero-dependency CLI over the `code` resource pack (generated by
[agentic-anything](https://github.com/THUQiXuan/agentic-anything)).

```bash
python code_cli.py pages
python code_cli.py page <page_id>
python code_cli.py --json search <keywords>
python code_cli.py apis
python code_cli.py fetch /some/path      # live same-origin GET
python code_cli.py form-curl 0           # curl template for form #0
```

All commands accept `--json` for machine-readable output.
Exit codes: 0 = success, 1 = error (message on stderr).
