# cs336-course CLI

Zero-dependency CLI over the `cs336-course` resource pack (generated by
[agentic-anything](https://github.com/THUQiXuan/agentic-anything)).

```bash
python cs336_course_cli.py pages
python cs336_course_cli.py page <page_id>
python cs336_course_cli.py --json search <keywords>
python cs336_course_cli.py apis
python cs336_course_cli.py fetch /some/path      # live same-origin GET
python cs336_course_cli.py form-curl 0           # curl template for form #0
```

All commands accept `--json` for machine-readable output.
Exit codes: 0 = success, 1 = error (message on stderr).
