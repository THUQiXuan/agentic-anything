# handbook CLI

Zero-dependency CLI over the `handbook` resource pack (generated by
[agentic-anything](https://github.com/THUQiXuan/agentic-anything)).

```bash
python handbook_cli.py pages
python handbook_cli.py page <page_id>
python handbook_cli.py --json search <keywords>
python handbook_cli.py apis
python handbook_cli.py fetch /some/path      # live same-origin GET
python handbook_cli.py form-curl 0           # curl template for form #0
```

All commands accept `--json` for machine-readable output.
Exit codes: 0 = success, 1 = error (message on stderr).
