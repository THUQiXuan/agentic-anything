Stop reading resources.

Agentic Anything turns any website, PDF, video, repository, or dataset into an agent-native pack — then into a resource agent you (or your agents) can question, call, and audit.

websitePDFvideo repodatasetsoftware ──▶ an agent you can talk to
Everything below is a replay of committed recordings … offline checks pass in CI … recorded real-model calls; zero calls in your browser

The same one command, five more shapes of resource

A repository, a book, a docs site, a NASA dataset, a scholarly paper — ask; get a cited, hash-pinned answer.

Do this to your own resources

Core install has zero runtime dependencies.
1 · install + agentify anything
pip install agentic-anything
$ agentic-anything agentify \
    https://cs336.stanford.edu/spring2025/ \
    --follow-docs 6 --follow-repos 1
2 · talk to it — or hand it to your agent
$ agentic-anything chat packs/cs336-course
$ agentic-anything mcp-config \
    packs/cs336-course --client claude
3 · or serve it to other agents
$ agentic-anything serve packs/* --enable-a2a
# OpenAI-compatible + A2A endpoints
What is real here
  • Packs are built from pinned, hash-verified sources.
  • Conversations were recorded once against a real model; citations re-verify offline.
  • Long runs drive a real stdio MCP subprocess; every JSON-RPC exchange is committed.
What this page does
  • The page is a replay. It replays committed recordings; no model runs in your browser.
  • Every citation chip opens the actual captured evidence.
  • Rebuild everything: python demos/build_demos.py
