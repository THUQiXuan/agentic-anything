Why the pack mattered
One interface across
three native formats.

        

        

          
BEFORE
Three incompatible inputs
Requests arrives as a 74-file repository, GISTEMP as a 147-year CSV table, and FAIR as a long scholarly paper. Their native structures do not match.

          
AGENT PACK
The same three tools
resource_info, search_resource, and read_unit served the repository, CSV, and paper in these runs, returning stable unit IDs, locators, and hashes.

          
AFTER
Three checked deliverables
Those calls fed a 21-step model run, a cross-pack climate/FAIR audit, and a deterministic code-impact trace; every published claim points back to a unit the agent read.

Boundary of this demo

            
    
              
  • “Replay” animates a checked-in recording; GitHub Pages is not secretly running a server.
    • 
              
  • The model run is a real one-time tool loop. Model text is not trusted until its evidence gate passes.
    • 
              
  • Deterministic runs demonstrate tool use and calculations, not hidden LLM reasoning.
    • 
              
  • All source access is read-only. No upstream repository, dataset, or paper is modified.
    • 
              
  • Captured code units can be truncated; the release ZIP hash authenticates the complete upstream snapshot.
    •
