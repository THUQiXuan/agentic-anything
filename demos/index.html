AGENT.md tells a human or agent where to start. agent-interface.json advertises machine-readable entry points. Search and read work offline; MCP exposes the same evidence to Codex and other hosts.

✓ Directly demonstrated

  • Five source types become valid resource-agent packs.
  • Every result carries unit, locator and content hash.
  • Generated CLIs run; MCP searches and reads multiple packs.
  • All displayed evidence comes from committed raw results.

Boundary of this demo

  • It measures deterministic capture and use—not LLM answer quality.
  • It does not claim perfect ingestion of every resource.
  • Read-only MCP reduces tool risk but cannot eliminate host prompt injection.
  • Human productivity still requires a separate user study.
