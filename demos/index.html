Honest quality boundary

This page separates what the deterministic demo directly proves from what still needs an LLM or a user study.

Demonstrated here

  • Five heterogeneous sources produce valid packs.
  • Queries return inspectable units and evidence.
  • Generated resource CLI executes successfully.
  • MCP searches and reads across multiple packs.
  • No credentials or model calls are required.

Not claimed by this demo

  • Perfect ingestion of every possible resource.
  • Generative answer quality or hallucination rate.
  • Human productivity improvement.
  • Recovery of every native website/software action.
  • Compatibility with every agent host.
